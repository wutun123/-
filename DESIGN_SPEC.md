## 七、交互原型说明

### 7.1 页面跳转关系

```
启动页 → 首页
           │
    ┌──────┼──────┬──────────┐
    │      │      │          │
  商品    分类   购物车     我的
    │              │          │
  详情            结算        订单
    │              │          │
  下单            支付        设置
```

### 7.2 核心交互流程

#### 购物流程
```
首页/搜索 → 商品详情 → 选择规格 → 加入购物车 →
购物车 → 确认订单 → 微信支付 → 支付成功 → 订单列表
```

#### 分销申请流程
```
我的 → 申请分销商 → 填写资料 → 提交 →
待审核 → 审核通过 → 分销中心
```

#### 佣金提现流程
```
分销中心 → 可提佣金 → 立即提现 →
选择金额 → 确认 → 微信分账 → 提现成功
```

## 九、前端开发对接说明

### 9.1 开发注意事项
1. 所有尺寸使用 rpx 单位（1rpx = 0.5px）
2. 图片资源需要 @2x、@3x 多倍图
3. 颜色值统一使用 6 位十六进制
4. 字体颜色注意对比度，确保可读性

### 9.2 切图标注
- 所有切图提供 Sketch/Figma 源文件
- 图标提供 SVG 矢量格式
- 复杂动效提供 Lottie JSON
